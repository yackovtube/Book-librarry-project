
## Env Setup
Install [`NodeJS`](https://nodejs.org/en/) and [`NPM`](https://nodejs.org/en/)

## Installation
run in project root directory

```bash
$ npm install
```

## Commands
### Run project
```bash
$ npm run serve
```
### Run unit test
```bash
$ npm run test
```
### Run coverage (unit test)
```bash
$ npm run covrage
```
